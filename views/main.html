About this thing

There's been a big increase in the focus on Typography as the basis for web design in the last few years. One of the trends that have emerged as a result is the use of Modular Scales for design (though typographic scales have obviously been in use for many years), made popular by the A List Apart article: More Meaningful Typography

This app is intended to help the process of improving typography on existing sites. If a site uses arbitrarily chosen font choices, then it can be difficult to force them onto a scale without making large changes. This app tests input font sizes against various commonly used modular scales, works out which one's the closest fit, and then matches your font sizes with the nearest equivalents on that scale.

More than anything else though, this is an experiment with Angularjs.
